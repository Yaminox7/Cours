C'est quoi un site web

Un site web ou site, est un ensemble de pages web reliées par des hyperliens et accessible par une adresse web. L'adresse web d'un site correspond en fait à l'URL d'une page web prévue pour être la première consultée : la page d'accueil. Un site est développé à l'aide de langages de programmation web (comme le HTML), puis hébergé sur un serveur web accessible via le ou n'importe quel réseau.

C'est quoi le langage HTML

Le langage HTML est un langage de balisage conçu pour créer des pages web. Sa version la plus récente est HTML5 de logo .Ce langage permet de structurer sémantiquement une page web, de mettre en forme du contenu, de créer des formulaires ou encore d'inclure des ressources multimédias dont des images, des vidéos... Il est directement compilé par le navigateur web sans avoir besoin de l'installer. Il est majoritairement joint de CSS pour les styles et de JS pour le rendre dynamique. L'extension d'un fichier HTML est .htm ou .html.
